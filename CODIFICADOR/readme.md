# 🔐 Lenguaje Secreto con Símbolos Personalizados

Este proyecto te permite codificar y decodificar mensajes usando un lenguaje secreto basado en símbolos extraños. Ideal para compartir mensajes misteriosos con tus amigos o crear códigos únicos.

## 🚀 ¿Cómo funciona?

- Cada letra del alfabeto (A-Z y espacio) tiene un símbolo único asignado.
- Puedes escribir un mensaje normal y convertirlo en un mensaje cifrado con símbolos.
- También puedes tomar un mensaje cifrado y traducirlo de vuelta a texto.

## 📦 Requisitos

Solo necesitas Python 3 para ejecutar este script. No hay dependencias externas.
